variable:
	Es un "objeto" el cual se le puede asignar un nombre y un valor. Por
ejemplo, el nombre de una variable podria ser altura y el valor 1,75.
Ademas se puede operar con ellas (operaciones matematicas), o se las
puede utilizar dentro de funciones-

String:
	son un conjunto de caracteres, por no decir letras, siempre 
van entre comillas simples (''), generalmente se las usa como palabras

Funciones:
	Una funcion es un conjunto de ordenes que se le da al editor 
de texto para que se comunique con la pc para realizar determinada
operacion (la que se crea con el codigo)
Argumento: lo que se le da a la funcion para que trabaje, utilice esa
informacion o datos, pueden ser variables o numeros, incluso puede
ser un argumento vac√¨o.
'return': es un comando podriamos decir, que hace que el output sea 
lo que le sigue despues de escribir return, hace que se termine la 
funcion. Si escribis una orden despues de una linea que tenga 'return'
no se va  a precesar/leer/como le quieras decir.

Declaraciones 'if':
	Es para crear operaciones que tengan una condicion. IF en ingles
es el si, se usa para los condicionales, entocnes en JS se utiliza
if (condicion que quieras) y se puede cumplir (true), no cumplir
(false), o la podes setear para que siempre se cumpla o nunca se cumpla.

Valores booleanos:
	son resultados que no son numericos ni strings, solo pueden
ser verdaderos o falsos.
